Expenses Manager (provisional name) is a personal expense and income tracking application. Currently in very early development stages, with basic functionality implemented but many components still under construction.

## 🚧 Project Status

This app is being developed as a personal project and learning exercise. Not recommended for production use or for managing real financial data at this stage.

### ✅ Currently Implemented
- Basic CRUD operations for transactions
- Connection to database
- Initial project structure
- First auth
- First charts
